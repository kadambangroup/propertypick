# PropertyPick Website

A modern, dynamic real estate website template built with HTML, CSS, and JavaScript.

## Features
- **Dynamic Listings**: Automatically generates property cards.
- **Search Filters**: Filter by City, Pincode, and Buy/Rent status.
- **Modern Design**: Warm, bright, and responsive interface.
- **Authentication**: Mock login with phone number and password.

## How to Run
Simply double-click `index.html` to open the website in your browser. No installation required.

## Customization
- **Add Cities**: Open `data.js` and add new city names to the `CITIES` array.
- **Change Colors**: Open `styles.css` and modify the colors in the `:root` section at the top.
- **Logo**: Replace the text in `index.html` with your `<img>` tag.

## Files
- `index.html`: Main structure.
- `styles.css`: Styling and design.
- `app.js`: Logic for search, navigation, and login.
- `data.js`: Data for cities and listings.
